## Installing

Clone the repo:

```
git clone https://github.com/HenrikLeutscher/FED2-CSSFrameworksCA-HenrikLeutscher.git
```

Navigate to the project folder:

```
cd FED2-CSSFrameworksCA-HenrikLeutscher
```

Install Dependencies:

```
npm install
```

## Running

To run the project with Tailwind CSS watching for changes:

```
npm run dev
```

To run the project:
Go live with liveServer extension in VSCode.

Open the project by using Live Server extension in VS Code.
